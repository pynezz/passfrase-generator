# Passfrasegenerator

En generator som genererer norske ord som høres korrekte ut, men som ikke eksisterer.

>[!WARNING] Work in progress
> Dette er en work in progress. Virker foreløpig kun i terminal.

## Test generatoren

1. Klon repoet
2. Kjør `parser.py`
